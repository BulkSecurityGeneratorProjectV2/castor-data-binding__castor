Quick Description

Castor is an Open Source data binding framework for Java[tm]. It's the shortest path between Java objects, XML documents and relational tables. Castor provides Java-to-XML binding, Java-to-SQL persistence, and more.

JDO extensions for the Castor XML code generator - Preview

I am very pleased to announce the immediate availability of the JDO extensions for the Castor XML code generator in form of a technology preview as part of the first snapshot release of Castor 1.3.1, which can be found at and downloaded from the Codehaus snapshot repository.
